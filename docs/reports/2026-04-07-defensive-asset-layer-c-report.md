## 观察

### 1. `USO/BNO` 提供的是商品 beta，不是默认防御属性
- C1/C2/C3 全样本收益都高于 A1/A2，但 MaxDD 和滚动最差收益都明显恶化。
- 这类提升主要来自商品上涨阶段，而不是更稳的风险关闭表现。
- 尤其 C1/C2 在 Phase 1 和 Phase 2 的高收益，明显带有能源商品行情驱动特征。

### 2. 2022 阶段收益转正，不代表更稳健
- C1/C2/C3 在 2022 阶段的收益明显优于 A1/A2：
  - C1：`8.18%`
  - C2：`12.14%`
  - C3：`5.68%`
- 但同一阶段的回撤也明显更深：
  - C1：`-12.40%`
  - C2：`-11.36%`
  - C3：`-11.64%`
  - 对比 A1/A2 仅 `-7.55% / -7.42%`
- 这说明商品暴露提供的是方向性收益补偿，不是更可控的防御波动。

### 3. 商品组的回撤持续时间明显拉长
- A1/A2 的最大回撤持续时间分别是 `66/63` 天。
- C1/C2/C3 则达到 `128/126/109` 天。
- 这对长期稳健目标很关键：恢复变慢，意味着防御层在组合受损后更难快速回到新高。

### 4. `BNO` 比 `USO` 略好，但仍然不是默认候选
- C2 在商品组里最像“较温和版本”：
  - CAGR 最高
  - MaxDD 低于 C1/C3
  - Phase 2 收益最好
- 但它依然显著弱于 A1/A2 的稳健性标准，特别是回撤和恢复期。
- 所以 C2 可以保留为观察型研究对象，但不能进入默认防御候选。

### 5. 双商品组合 C3 最差
- C3 的 MaxDD 达到 `17.9%`，远高于 A1/A2，也高于 C1/C2。
- 63 日最差滚动收益 `-10.68%`，是当前所有商品组里最差。
- `USO+BNO` 叠加没有带来分散化，反而更像把商品风险放大。

## 按稳健门槛判断
按实验计划里的“稳健优先”门槛：
- C1/C2/C3 都不能进入默认防御配置候选。
- 它们的收益改善不具备跨阶段稳健性，且以更深回撤和更长恢复期为代价。
- `USO/BNO` 应被归类为商品观察标的，而不是防御资产。

## Layer C 结论
1. **`USO` 与 `BNO` 都不应进入默认防御篮子。**
2. **`BNO` 相比 `USO` 略稳，但仍只适合作为商品观察组研究对象。**
3. **`USO+BNO` 双商品组合应停止继续作为候选主线。**
4. **若长期目标是稳健，商品类资产只能作为显式研究扩展，不能进入默认 risk-off / idle defensive 路径。**
